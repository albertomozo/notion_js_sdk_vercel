API - Mis cursos

Version V.1.1.0

Esta API permite realizar operaciones con cursos que estan almacenados en esta Base de datos de Notion . A continuación se listan los endpoints disponibles:

la url base es : https://notion-js-sdk-vercel.vercel.app

facilmente.
Método Ruta Descripción Enlace
GET /cursos Devuelve todos los cursos pruebalo
GET /cursosweb Devuelve todos los cursos que estan marcados para ver en web en bruto, tal y como los almacena Notion pruebalo
GET /cursosweb2 Devuelve todos los cursos que estan marcados para ver en web de manera más amigable, para poder procesarlos pruebalo
GET /cursos/:Nombre Devuelve el curso con Nombre = :Nombre pruebalo
GET /categorias Devuelve todas las categorias pruebalo
GET /categorias/cursos Devuelve todas los nombres,id de pagina , enlaces de los cursos, agrupados por Categorias pruebalo
